# 5/15 김종혁 T 강의 정리
우리가 흔히 쓰는 어플리케이션을 실행시켜주는것을 운영체제(OS, Operating System)이라고 한다. 우리가 쓰는 Windows 운영체제 안에서 또다른 운영체제를 실행 시킬수 있다. 

## VM WARE
VM (Virtual Machine)안에 IOS, Linux 등 다양한 운영체제를 다운로드 할 수 있고 내 PC 안에 또다른 PC를 설치하는 형태라고 생각하면 된다.

리눅스 설정을 하였다.

# 리눅스 기본 명령어 정리
리눅스를 사용할때 굉장히 자주 타이핑 하는 기본 명령어들이다.

모든 명령어는 명령어 뒤에 --help 옵션을 주면 자세한 사용 방법이 나온다. 예를 들어 ls 명령어의 자세한 사용 방법과 모든 옵션을 알고 싶으면 ls --help를 입력하면 된다.

## pwd (print working directory)
현재 작업중인 디렉토리 정보를 출력한다.

```bash
$ pwd
/home/itholic
```
## cd (change directory)
경로 이동

절대 경로와 상대 경로로 이동 가능하다.
```
$ cd /home/itholic/mydir
$ pwd
/home/itholic/mydir


$ cd ..
$ pwd
/home/itholic
```
## ls (list)
```ls``` 명령어는 현재 폴더의 하위 디렉토리의 리스트를 보여줍니다.
```bash
$ ls
testfile1  testfile2  testfile3


$ ls -l
(long을 의미, 파일권한(drwxr/xr/x/r 등), 생성 날짜 등 파일에 대한 자세한 정보를 출력합니다.)
total 0
-rw-r--r-- 1 itholic 197121 0 11월  6 22:08 testfile1
-rw-r--r-- 1 itholic 197121 0 11월  6 22:08 testfile2
-rw-r--r-- 1 itholic 197121 0 11월  6 22:08 testfile3


$ ls -a 
(all, 숨겨진 모든 파일 등을 보여줍니다. 숨겨진 파일은 앞에 . 마침표가 붙습니다.)
./  ../  testfile1  testfile2  testfile3


$ ls -al
(숨긴 파일을 파일 권한, 크기등 모든 데이터를 출력합니다.)
total 4
drwxr-xr-x 1 itholic 197121 0 11월  6 22:08 ./
drwxr-xr-x 1 itholic 197121 0 11월  6 22:08 ../
-rw-r--r-- 1 itholic 197121 0 11월  6 22:08 testfile1
-rw-r--r-- 1 itholic 197121 0 11월  6 22:08 testfile2
-rw-r--r-- 1 itholic 197121 0 11월  6 22:08 testfile3
```
## cp (copy)
파일 혹은 디렉토리를 복사

디렉토리를 복사할때는 -r 옵션을 주어야함
```bash
$ ls
testdir/  testfile


$ cp testfile1 testfile_cp
$ ls
testdir/  testfile  testfile_cp


$ cp -r testdir testdir_cp
$ ls
testdir/  testdir_cp/  testfile  testfile_cp
```
## mv (move)
파일 혹은 디렉토리 이동, 실제로 원하는 위치로 이동할때도 사용하지만, 이름을 변경하는 용도로도 사용한다.

cp와는 달리 디렉토리를 이동할때도 별다른 옵션이 필요 없다.
```bash
$ ls
testdir/  testfile


$ mv testfile testfile_mv
$ ls
testdir/  testfile_mv


$ mv testfile_mv testdir/
$ ls
testdir/


$ ls testdir/
testfile
```
## mkdir (make directory)
디렉토리 생성, -p 옵션을 주면 하위 디렉토리까지 한 번에 생성 가능
```bash
$ ls
testfile


$ mkdir testdir
$ ls
testdir/  testfile


$ mkdir -p a/b/c/d/e/
$ ls -R a/
a/:
b/

a/b:
c/

a/b/c:
d/

a/b/c/d:
e/

a/b/c/d/e:
```
## rm (remove)
파일이나 디렉토리를 삭제, 디렉토리를 삭제할때는 r 옵션을 주어야 한다.

-f 옵션을 주면 사용자에게 삭제 여부를 묻지 않고 바로 삭제한다. 디렉토리를 삭제할 때에는 하위 디렉토리까지 모두 삭제되므로 유의하자.
```bash
$ ls
testdir/  testfile1  testfile2


$ rm -f testfile1
$ ls
testdir/  testfile2


$ rm -rf testdir/
$ ls
testfile2
```
## touch
파일이나 디렉토리의 최근 업데이트 일자를 현재 시간으로 변경한다. 최근 업데이트 일자는 ```ls -l``` 명령을 통해 확인할 수 있다.

아래 예제에서 '11월 6 22:08' 이라고 쓰여진 부분이다. 파일이나 디렉토리가 존재하지 않으면 빈 파일을 만든다.
```bash
$ ls -l
total 0
-rw-r--r-- 1 itholic 197121 0 11월  6 22:08 testfile1


$ touch testfile1
$ ls -l
total 0
-rw-r--r-- 1 itholic 197121 0 11월  6 22:43 testfile1


$ touch testfile2
$ ls -l
total 0
-rw-r--r-- 1 itholic 197121 0 11월  6 22:43 testfile1
-rw-r--r-- 1 itholic 197121 0 11월  6 22:44 testfile2
```
## cat (concatenate)
cat 명령은 활용 방법이 꽤나 다양하다. 단순히 파일의 내용을 출력할 수도 있고, 파일 여러개를 합쳐서 하나의 파일로 만들 수도 있다.

그리고 기존 한 파일의 내용을 다른 파일에 덧붙일수도 있다. 새로운 파일을 만들때에도 사용된다.

file1, file2, file3 파일에는 각각 간단하게 숫자 1,2,3 이 적혀있다.
```bash
$ cat file1
1


$ cat file2
2


$ cat file3
3


$ cat file1 file2 > file1_2
$ ls
file1  file1_2  file2  file3


$ cat file1_2
1
2


$ cat file1 >> file2
$ cat file2
2
1


$ cat > file4
hello
world
(작성이 끝나면 ctrl +d 로 파일 저장)


$ cat file4
hello
world
```

## find
특정 파일이나 디렉토리를 검색한다 사용법이 앞의 명령어들에 비해 살짝 복잡하므로, 기본 사용법을 언금하자면 다음과 같다.
```bash
find [검색경로] -name [파일명]
```
파일명은 직접 풀 네임을 입력해도 되지만, 다음 예제처럼 특정 조건으 적용해 검색할수도 있다.
```bash
$ ls
dir1/  dir3/  file1  file3  picture1.jpg  picture3.jpg
dir2/  dir4/  file2  file4  picture2.jpg  picture4.jpg


$ find ./ -name 'file1'
./file1


$ find ./ -name "*.jpg"
./picture1.jpg
./picture2.jpg
./picture3.jpg
./picture4.jpg
```
확장자가 .jpg인 파일을 찾았다. 하지만 여기서 그치지 않고, 확장자가 .jpg인 파일만 찾아서 바로 삭제할수도 있다.

exec 옵션을 사용해 다음과 같이 처리하면 된다.
```
$ find ./ -name "*.jpg" -exec rm {} \;
$ ls
dir1/  dir2/  dir3/  dir4/  file1  file2  file3  file4
